Подготовка среды разработки для WEB
  1. Среда выполнения на уровне сервера: Apache+PHP+MySQL. Самый простой вариант заключается в установке сборки XAMPP
  2. Среда непосредственного кодирования (среда разработки). Из IDE одним из удобных является NetBeans for PHP
  3. Среда выполнения клиентской части Web-приложения:
    • Для запуска/отладки клиентской части WEB-приложений можете использовать браузеры Google Chrome, Firefox.
    • Для полноценной отладки на браузере Firefox установите плагин FireBug (JS консоль и прочие вкусности).
    • Для интеграции дизайна понадобится плагин PixelPerfect (и для Chrome, и для Firefox)

  • Информация по основным моментам в xHTML
